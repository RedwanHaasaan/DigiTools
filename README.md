# 🛒 Digi-Tools

A modern, responsive e-commerce platform for showcasing and purchasing digital tools and products. Built with cutting-edge web technologies to provide users with a seamless shopping experience.

---

## 📋 Description

**Digi-Tools** is a feature-rich web application designed to simplify the process of browsing, selecting, and purchasing digital products. Whether you're looking for software tools, plugins, or digital services, our platform offers an intuitive interface with smooth navigation, real-time cart management, and comprehensive product information. With a focus on user experience and modern design principles, Digi-Tools brings your digital marketplace to life.

---

## 🛠️ Technologies Used

- **Frontend Framework:** React 19.2.4
- **Build Tool:** Vite 8.0.1
- **Styling:** Tailwind CSS 4.2.2
- **UI Components:** DaisyUI 5.5.19
- **Icons:** FontAwesome & Lucide React
- **Notifications:** React Toastify 11.0.5
- **Runtime:** Node.js (Module Type)

---

## ✨ Features

### 🎯 1. **Dynamic Product Showcase**
Browse through an extensive catalog of digital products with beautifully designed product cards. Each product displays detailed information including pricing, descriptions, and specifications. The product grid is fully responsive, adapting seamlessly across desktop, tablet, and mobile devices.

### 🛍️ 2. **Smart Shopping Cart**
Add or remove products from your cart with real-time updates. Get instant notifications for cart operations, view your total items in the navigation bar, and manage your purchases efficiently. The cart persists during your session, allowing you to shop without losing track of your selections.

### 💰 3. **Pricing & Statistics Dashboard**
Explore our competitive pricing plans and view key statistics about our platform including customer satisfaction, trusted users, and successful transactions. Display aggregate metrics that build trust and showcase the value of our digital tools ecosystem.

## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd digi-tools
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
